# Quiz Website

This is a simple quiz website built using HTML, CSS, and JavaScript. It presents multiple-choice questions to users and evaluates their answers to provide a final score.

## Features

- Multiple-choice questions with selectable options.
- Submit button to submit answers and progress to the next question.
- Score calculation and display.
- Responsive design for optimal viewing on various devices.

## Technologies Used

- **HTML**: Provides the structure and content of the webpage.
- **CSS**: Styles the webpage for improved aesthetics and user experience.
- **JavaScript**: Implements interactivity and logic for the quiz functionality.
